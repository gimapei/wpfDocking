Remarks

            This method cannot be overridden by derived classes. A
            derived class should override the [M:Append(LoggingEvent[])] method
            which is called by this method.
            

            The implementation of this method is as follows:
            
  • 
            		Checks that the severity of the loggingEvents
            		is greater than or equal to the Threshold of this
            		appender.
  • 
            		Checks that the IFilter chain accepts the 
            		loggingEvents.
            		
  • 
            		Calls [M:PreAppendCheck()] and checks that 
            		it returns true.

            If all of the above steps succeed then the loggingEvents
            will be passed to the [M:Append(LoggingEvent[])] method.
