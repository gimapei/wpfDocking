Remarks

            Helper method to render a LoggingEvent to 
            a string. This appender must have a Layout
            set to render the loggingEvent to 
            a string.
            
If there is exception data in the logging event and 
            the layout does not process the exception, this method 
            will append the exception text to the rendered string.
            

            Use this method in preference to [M:RenderLoggingEvent(LoggingEvent)]
            where possible. If, however, the caller needs to render the event
            to a string then [M:RenderLoggingEvent(LoggingEvent)] does
            provide an efficient mechanism for doing so.
