Remarks

            The message is formatted using the String.Format method. See
            [M:String.Format(string, object[])] for details of the syntax of the format string and the behavior
            of the formatting.
            

            This method does not take an Exception object to include in the
            log event. To pass an Exception use one of the [M:Fatal(object)]
            methods instead.
