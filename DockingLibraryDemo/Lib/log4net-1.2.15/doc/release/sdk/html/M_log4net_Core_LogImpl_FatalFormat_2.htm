Remarks

            The message is formatted using the [M:String.Format(IFormatProvider, string, object[])] method. See
            String.Format for details of the syntax of the format string and the behavior
            of the formatting.
            

            The string is formatted using the InvariantCulture
            format provider. To specify a localized provider use the
            [M:FatalFormat(IFormatProvider,string,object[])] method.
            

            This method does not take an Exception object to include in the
            log event. To pass an Exception use one of the [M:Fatal(object)]
            methods instead.
