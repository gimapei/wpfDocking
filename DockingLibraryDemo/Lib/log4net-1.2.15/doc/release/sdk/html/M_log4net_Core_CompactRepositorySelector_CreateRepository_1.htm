Remarks

            The ILoggerRepository created will be associated with the repository
            specified such that a call to [M:GetRepository(string)] with the
            same repository specified will return the same repository instance.
            

            If the named repository already exists an exception will be thrown.
            

            If repositoryType is null then the default 
            repository type specified to the constructor is used.
